üîßTroubleshooting

Common Issues

Button says "Merge PDFs" but I selected "PDF to Images"
This has been fixed! The app now properly detects mode changes. If you still see this, try refreshing the page.
Files won't upload
Make sure you're uploading the correct file types:
  • PDF Tools: Only PDF files (.pdf)
  • Watermark Inserter: PDF files and images (.jpg, .jpeg, .png)
Browser freezes with large files
Very large PDFs may take time to process. Wait for the progress bar to complete. If your browser crashes, try processing fewer or smaller files at once.
Watermark preview doesn't show
The preview loads the watermark image from GitHub. Ensure you have an internet connection when first loading the page.
